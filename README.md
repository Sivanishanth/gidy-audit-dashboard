# Gidy Security Audit Dashboard

A full-stack Security Audit Dashboard built using React, Node.js, Express, and MongoDB.

## Features

- View security audit logs
- Search logs
- Filter by Severity, Role, and Region
- Pagination
- Upload JSON log files
- Responsive UI

## Tech Stack

### Frontend
- React
- Vite
- Axios
- CSS

### Backend
- Node.js
- Express.js
- MongoDB Atlas
- Mongoose

## Installation

### Backend

```bash
cd backend
npm install
node server.js
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
